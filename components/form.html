Form

A form displays a set of related user input fields in a structured way.

Basic Form Fields

A field is a form element containing a label and an input

We'll never share your email with anyone 
                            
                            
    <div class="form__field field">
        <label class="field__label" for="email">Email</label>
        <input class="field__input" id="email" type="email" placeholder="Enter email"
            autocomplete="email">
        <small class="field__text text-muted">We'll never share your email with anyone</small>
    </div>
                            
                        
                            
                            
    <div class="form__field field">
        <label class="field__label" for="password">Current Password</label>
        <input class="field__input" id="password" type="password" placeholder="Enter password"
            autocomplete="current-password">
    </div>
    <div class="form__field field">
        <label class="field__label" for="password">New Password</label>
        <input class="field__input" id="password" type="password" placeholder="Enter password"
            autocomplete="new-password">
    </div>
                            
                        
                            
                            
    <div class="form__field field">
        <label class="field__label" for="date">DOB</label>
        <input class="field__input" type="date" id="date">
    </div>
    <div class="form__field field">
        <label class="field__label" for="file">File input</label>
        <input class="field__input" type="file" id="file" multiple>
    </div>

    </div>

Form Field States

                            
                            
    <div class="form__field field field--readonly">
        <label class="field__label " for="text">Read-only input</label>
        <input class="field__input" id="text" type="text" value="Read only" readonly>
    </div>
                            
                        
                            
                            
    <div class="form__field field field--disabled">
        <label class="field__label" for="text">Disabled input</label>
        <input class="field__input" id="text" type="text" disabled>
    </div>
This is a required field. 
                            
                            
    <div class="form__field field field--required">
        <label class="field__label" for="text">Required input</label>
        <input class="field__input" id="text" type="text" required>
        <small class="field__text">This is a required field.</small>
    </div>

Form Field Sets

A set of fields that can appear grouped together

Custom checkbox 
                            
                            
    <fieldset class="form__field field--set field--horizontal field--checkbox">
        <legend class="field__label">Custom checkbox</legend>
        <div class="checkbox--container">
            <input type="checkbox" id="option-1" name="option-1">
            <label class="field__label" for="option-1">Option 1</label>
        </div>
        <div class="checkbox--container">
            <input type="checkbox" id="option-2" name="option-2">
            <label class="field__label" for="option-2">Option 2</label>
        </div>
        <div class="checkbox--container">
            <input type="checkbox" id="option-3" name="option-3">
            <label class="field__label" for="option-3">Option 3</label>
        </div>
    </fieldset>
